\section{Introduction}
Ada draws a clear syntactic separation between declarations, which introduces names for entitites that will be used in the program, and statements, which perform the processing. The areas in the program where declarartions my appear ar knonw as \textbf{declarative regions}.\\

In any suprogram, the section between the \verb|is| and the \verb|begin| is \textbf{delcarative region}. You can have varibles, constants, types, inner subprograms, and other entities here. 

\begin{lstlisting}
With Ada.Text_IO; use Ada.Text_IO;

procedure Main is 
    X: integer; 
begin
    X := 0;
    Put_Line("The initial value of X is " & Integer'Image(X));

    Put_Line("Performing operation on X ... ");

    X := X + 1;

    Put_Line("The value of X now is " & Integer'Image(X));
end Main;
\end{lstlisting}

In this particular case, the declaration region is betweeen line 3 and 5 in which there is a declaration of a single Variable of integer
